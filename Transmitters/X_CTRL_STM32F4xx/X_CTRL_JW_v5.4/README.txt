# Arduino for STM32F4xx

# https://github.com/FASTSHIFT/Arduino-For-Keil

注意{ 
	此版本的标准外设库默认晶振频率是25MHz;
	若板载晶振为8MHz,需要将stm32f4xx.h里的HSE_VALUE的定义改为 ((uint32_t)8000000),
	以及system_stm32f4xx.c里的PLL_M定义 25 改为 8 !
}
